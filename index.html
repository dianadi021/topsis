Metode TOPSIS (Technique For Others Preference by Similarity to Ideal Solution)

informasi

Website ini merupakan sebuah aplikasi yang berfungsi sebagai alat perhitungan Metode TOPSIS (Technique For Others Preference by Similarity to Ideal Solution). Website ini dibuat dengan tujuan untuk mempermudah dalam melakukan proses perhitungan metode tersebut dan juga sebagai salah satu syarat wajib lulus dari Program Studi Informatika Universtas AMIKOM Yogyakarta tahun 2023. Adapun juga website ini terbuat karena terinspirasi dari salah satu website yaitu Decision Radar Ez-TOPSIS dan saran dari bapak pembimbing penulisan naskah skripsi yaitu; Bapak Bayu Setiaji, M.Kom. Website ini dibangun/dibuat dengan menggunakan Native Programming Website, yang dimana menggunakan Framework Tailwind CSS dan perumusan/perhitungan rumus dari website Buku Informatika.
